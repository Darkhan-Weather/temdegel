Google Drive синк

Өгөгдөл таны Google Drive-ийн ХСТИ Систем folder-д ангилсан 3 файлд хадгалагдана.

Backup / Сэргээх

Хадгалах бүрт автоматаар backup үүснэ (сүүлийн 10-г хадгална). Алдаа гарвал өмнөх хувилбар руу буцаж болно.

JSON импорт

JSON файлаас тэмдэглэл нэмэх — давхардсан бүртгэл орохгүй (id-р шалгана)

Заалт засварлах

Заалтын нэр, дугаарыг засаад "Хадгалах" дарна. 2027 он болоход шинэ заалт нэмж эсвэл хасаж болно.

Давтагддаг ажлууд (Шаблон)

Өдөр бүр давтагддаг ажлыг шаблон болгож хадгалаарай. Хяналтын самбар дээр нэг товшилтоор өнөөдрийн бүртгэл үүснэ.
